# NayePankh Foundation - Premium UI/UX Redesign 🚀

A modern, highly responsive, and premium dark-theme redesign of the NayePankh Foundation website. This project focuses on improving user experience (UX), maximizing donation conversion rates through strategic UI elements, and providing a seamless mobile-first experience.

## 🌐 Live Demo
Check out the live website here: **[https://nazishahmad59.github.io/NAYEPANKH-NGO/donate.html]**

## 🎯 Objective
The primary goal of this redesign was to transform the existing NGO website into a modern, trustworthy, and visually appealing platform. By introducing a dark theme with glassmorphism effects, smooth animations, and better typography, the website now reflects the immense value and impact of the NGO.

## ✨ Key Features

- **Premium Dark Theme**: A visually striking dark mode interface using `#0B1121` and `#ff7300` (NayePankh Orange) for high contrast and readability.
- **Glassmorphism UI**: Modern frosted-glass effects on the navigation bar and floating elements.
- **Immersive Mobile Menu**: A full-screen, fluid slide-in mobile navigation menu designed for modern smartphone users.
- **Smooth Scroll Animations**: Elements smoothly fade and slide up as the user scrolls, built using the `IntersectionObserver` API for maximum performance.
- **Dynamic Number Counters**: Animated statistics (e.g., "2,00,000+ People Fed") that count up when scrolled into view.
- **Auto-playing Touch Slider**: A custom-built, lightweight image slider with auto-play functionality, manual controls, and mobile swipe/touch support.
- **High-Conversion Donate Buttons**: Animated CTA (Call to Action) buttons with pulse effects, glowing auras, and hover scale animations to encourage donations.

## 🛠️ Tech Stack

- **HTML5**: Semantic and accessible markup.
- **Tailwind CSS**: Utility-first CSS framework (via CDN) for rapid and responsive styling.
- **Vanilla JavaScript**: Pure ES6 JavaScript for DOM manipulation, sliders, and scroll animations (No jQuery/Heavy libraries).
- **Google Fonts**: 'Plus Jakarta Sans' for a clean, modern, and geometric typographic feel.
- **FontAwesome**: Scalable vector icons for social media, UI elements, and trusted badges.
